Overview

1. Introduction

This Privacy Policy describes how Network Meter ("the Extension"), developed by AZHAR (publisher: AZ&G), handles information when you install and use the extension in Google Chrome or Microsoft Edge.

Network Meter's sole purpose is to measure and display your real-time internet download and upload speed in the browser toolbar. It tracks daily data usage locally and allows you to run manual or automatic speed tests against Cloudflare's public speed infrastructure.

By installing or using the Extension, you agree to this Privacy Policy. If you do not agree, please uninstall the Extension.

Data

2. What Data Is Collected

Network Meter collects no personal data. The following table summarises every category of information and how it is handled:

Data Type Collected? Details
Personal identity Never Name, email, age, address — none of this is accessed or stored.
Browsing history Never URLs, page titles, or visited sites are never read or recorded.
Page content Never The extension reads only the Content-Length header to count bytes. No page text, images, or form data is accessed.
Speed readings Local only Download/upload Bps values are stored in chrome.storage.local on your device. They never leave your browser.
Daily usage totals Local only Byte counts for the current day are stored locally and reset each day. Never transmitted.
User preferences Local only Settings such as test duration and auto-interval are stored locally using chrome.storage.local.
IP address / location Never The extension does not access, log, or transmit your IP address or geographic location.
Authentication data Never Passwords, credentials, tokens — never accessed.
Permissions

3. Permissions Explained

The Extension requests the following browser permissions. Each permission is used strictly for the extension's core functionality and nothing else.

  • storage Stores speed readings, daily usage totals, and user settings locally on your device using chrome.storage.local. No data is synced to the cloud or transmitted anywhere.
  • webRequest Reads only the Content-Length response header and upload request body sizes to calculate real-time download and upload byte rates. No request URLs, content, or user data are accessed or stored.
  • alarms Schedules the periodic auto-speed-test timer at your configured interval (default: every 10 seconds). Used only to keep the toolbar badge updated in the background.
  • offscreen Creates an offscreen document to run the background auto-sampling scheduler, allowing speed tests to continue when the popup is closed. No user data is processed in this document.
  • <all_urls> Required by webRequest to intercept network response headers across all websites in order to measure download and upload speeds. The extension reads only byte-size metadata — never content, text, or URL paths.
External Services

4. Third-Party Services

Network Meter performs speed tests using Cloudflare's public speed measurement endpoints:

speed.cloudflare.com/__down — used to download test payload bytes.
speed.cloudflare.com/__up — used to upload randomly generated bytes.

These requests are made solely to measure network throughput. The payloads sent during upload tests are randomly generated binary data — they contain no personal information, no browsing data, and no identifiers.

Cloudflare's own privacy policy governs their handling of these requests. You can review it at cloudflare.com/privacypolicy.

The Extension does not use any analytics services, advertising networks, crash reporting tools, or any other third-party SDKs.

Storage

5. Data Storage & Retention

All data produced by the Extension is stored exclusively in chrome.storage.local — a browser-managed, device-local storage area that is never synchronized to any server or cloud account.

The following data is retained locally:

  • Last 60 speed samples (ring buffer) — overwritten continuously
  • Today's download and upload byte totals — reset each calendar day
  • Last manual speed test result (Mbps values and timestamp)
  • User settings (test duration, auto interval, payload size, enabled state)

All locally stored data is permanently deleted when you uninstall the Extension. You can also clear it manually at any time from the Extension's Settings page using Chrome's "Clear data" option.
